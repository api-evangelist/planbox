# PlanboxThis is a repo for managing machine readable artifacts for the [Planbox APIs](http://www.planbox.com). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [Planbox APIs](http://www.planbox.com).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](http://www.planbox.com)- [Developer](http://www.planbox.com)- [Twitter](https://twitter.com/PlanboxInc)- [GitHub](https://github.com/Planbox)The [APIs.json](https://github.com/api-evangelist/planbox/blob/master/apis.json) index provides links to all the relevant links for the Planbox APIs, including the OpenAPI and Postman if they are present.
